# Moyu Visual Novel Framework

The official visual novel development framework for Moyu Engine. For documentation, see [momoyu.ink/start/intro](https://momoyu.ink/start/intro).

[中文](README.md)

## Requirements

- Node.js 22+

## Install Dependencies

```bash
npm install
```

## Install and Update Engine

```bash
npm run engine:update
```

## Development

Start the development server:

```bash
npm run dev
```

Then start the engine in another terminal for preview. Using native application:

```bash
npm run engine:native
```

Or preview in browser:

```bash
npm run engine:web
```

Both support hot reload.

## Build

```bash
npm run build
```
